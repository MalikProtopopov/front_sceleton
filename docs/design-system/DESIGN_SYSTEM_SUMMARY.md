## THE VISUAL DIRECTION

### Colors

```
Primary Brand: Black (#000000) + Hot Pink (#FF006E)
Secondary Accent: Electric Blue (#0080FF)
Neutrals: Gray scale (F9FAFB to 111827)
Success/Status: Green (#4ade80), Yellow (#f59e0b), Red (#ef4444)
```

### Typography

```
Headlines: Poppins Bold (variable font, 64px-32px scale, animated)
Body: Inter Regular (16px, line-height 1.6)
Mono: IBM Plex Mono (for metrics, code snippets)
```

### Motion

```
Fast (hovers): 150ms ease-out-cubic
Normal (transitions): 250-300ms ease-out
Scroll reveals: 400-600ms ease-out (fade + slide)
Major animations: 800-1200ms (count-ups, morphs)
```

### Key Elements

- **Hero**: Large expressive typography + 3D object (Spline) + CTA
- **Process**: "Assembly line" style (Discovery → Dev → Launch → Iterate)
- **Proof**: Metrics prominently displayed, counting up on scroll
- **Case Studies**: Editorial layout + timeline + metrics + testimonial
- **Motion**: Kinetic, functional (not frivolous)
- **Conversion**: Clear CTAs, form-friendly, founder-focused language
